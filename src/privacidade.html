Quem é o responsável pelos seus dados

Este site é mantido em nome do Dr. Rodrigo José Barros Rodrigues Leite (MÉDICO · CRM-PE 15162), que atua como controlador dos dados pessoais coletados aqui, nos termos da Lei Geral de Proteção de Dados (Lei nº 13.709/2018).

Contato para questões de privacidade: WhatsApp (81) 99438-5112.

Quais dados coletamos

Este site é uma landing page estática. Nenhum dado é armazenado em banco de dados próprio. Os únicos dados pessoais que processamos são:

  • Nome, telefone e e-mail (opcional), quando você preenche o formulário de contato. Esses dados são enviados diretamente ao WhatsApp do Dr. Rodrigo junto com a mensagem que você envia.
  • Mensagens iniciadas via botão "Agendar consulta no WhatsApp", que redirecionam para o WhatsApp com texto pré-preenchido.

Não usamos cookies de rastreamento, pixels de publicidade nem ferramentas de analytics neste momento. Se essas ferramentas forem adicionadas no futuro, esta política será atualizada e um aviso será apresentado.

Para que usamos seus dados

Seus dados são usados exclusivamente para:

  • Entrar em contato com você via WhatsApp para agendar consulta.
  • Responder dúvidas administrativas sobre o atendimento (horários, localização, formas de pagamento).

Não enviamos comunicações de marketing, não compartilhamos seus dados com terceiros para fins promocionais e não utilizamos seus dados para análises estatísticas sem a sua autorização explícita.

Base legal para o tratamento

O tratamento dos seus dados é fundamentado em:

  • Consentimento (art. 7º, I da LGPD) — ao preencher e enviar o formulário, você consente com o envio dos seus dados via WhatsApp.
  • Execução de contratação a pedido do titular (art. 7º, V) — para viabilizar o agendamento da consulta solicitada.

Compartilhamento com terceiros

Ao enviar o formulário ou clicar em "Agendar consulta no WhatsApp", você é redirecionado para o WhatsApp (Meta Platforms, Inc.), que processa a mensagem conforme sua própria política de privacidade: whatsapp.com/legal/privacy-policy-br.

Dados de prontuário médico e informações clínicas que você eventualmente compartilhar em consulta são armazenados conforme as normas do Conselho Federal de Medicina e do Código de Ética Médica, separadamente deste site.

Seus direitos (LGPD)

Como titular dos dados, você pode a qualquer momento solicitar:

  • Confirmação da existência de tratamento de seus dados.
  • Acesso aos seus dados.
  • Correção de dados incompletos, inexatos ou desatualizados.
  • Anonimização, bloqueio ou eliminação de dados desnecessários ou tratados em desconformidade com a LGPD.
  • Portabilidade dos dados.
  • Revogação do consentimento.

Para exercer qualquer desses direitos, entre em contato pelo WhatsApp (81) 99438-5112.

Segurança

Este site é hospedado com HTTPS obrigatório e não armazena dados em servidores próprios. As mensagens enviadas via WhatsApp trafegam com criptografia de ponta a ponta conforme o protocolo do próprio WhatsApp.

Alterações nesta política

Esta política pode ser atualizada periodicamente. A data da última atualização aparece no topo deste documento. Alterações materiais serão sinalizadas no próprio site.
